
# Тестовое задание: Минимальный кастомный объект.

## Задача

Создать класс AstraMarker, который будет:

1. Являться кастомным объектом NanoCAD (McCustomBase) с уникальным GUID и именем ASTRA_MARKER.
2. Рисовать круг с заданным радиусом (например, 1.0) в локальной системе координат объекта.
3. Поддерживать сохранение и загрузку.

## Требования

- Объект должен появляться в чертеже после команды
- Не требуется грипсов или интерактивного редактирования.

## Проверка

- Сборка загружается.
- Команда добавления объекта создаёт круг.
- Сохранение и повторное открытие файла сохраняют радиус.

## Стенд разработки

Использовала nanoCad 26 версии + Net 6.
